# VTU Examination Result System (Simple Prototype)
This project is part of assignment.
This is a simple VTU Result lookup system created for assignment.

## Files included
1. index.html – user input page
2. result.php – displays sample results (no database)
3. readme.md – documentation

## How to run
- Keep files inside htdocs folder (XAMPP)
- Start Apache server
- Open http://localhost/vtu/index.html
- Enter USN (example: 1RV17CS001)

- Demonstrate SDLC process
- Link Jira tasks to GitHub commits
